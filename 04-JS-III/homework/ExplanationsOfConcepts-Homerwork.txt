Arrays: Es una lista de diversos tipos de datos. Son como contenedores de almacenamiento para colecciones de datos. 
Para construir un array, primero declaramos una variable y le asignamos [], luego podemos agregar al contenedor una lista de diversos tipos de datos como cadenas, números o booleanos, separados cada elemento con una coma.
